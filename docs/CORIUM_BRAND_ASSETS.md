## Motivo

La guía recibida es una composición rasterizada. Extraer transparencia, vectorizar curvas, reconstruir degradados o separar capas produciría una interpretación, no una copia fiel. Para preservar la identidad, esas variantes quedan pendientes de recibir como originales exportados desde diseño.
